# assets

Brand assets used by the documentation.

## logo.png

The top-level [README](../README.md) references `assets/logo.png`. Save the StackRef
logo here with that exact filename:

```
assets/logo.png
```

Notes:
- The provided logo has a dark navy background baked in. It will look great on
  GitHub's dark theme; on light theme it renders as a dark rectangle. If you want it
  to look clean on both, also export a transparent-background version and either swap
  it in or use a `<picture>` element in the README with light/dark variants.
- A horizontal lockup around ~420px wide displays well at the README's current size.
